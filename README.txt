DESCRIPCIÓN

Huevobit es un juego blockchain descentralizado multiplataforma que combina mecánicas de juego inmersivas con tecnología Web3. Desarrollado con Flutter, ofrece una experiencia unificada en iOS, Android, Web y Desktop.

TECNOLOGÍAS IMPLEMENTADAS

Framework Principal

- Flutter SDK 3.24.3
- Dart 3.5.3

Blockchain & Web3

- web3dart 3.0.1
- bip39 1.0.6
- blockchain_utils 2.1.2

Características Técnicas

- Multiplataforma nativa
- Gestión descentralizada de activos
- Contrato inteligente verificable
- UI adaptativa y responsive

INSTALACIÓN

Prerrequisitos
Flutter >= 3.24.0
Dart >= 3.5.0

Proceso de Instalación

1. git clone https://github.com/Kanopola/Huevobit/
2. cd kanopola
3. flutter pub get
4. flutter run

Configuración Blockchain

1. Configurar wallet en lib/nido.dart
2. Actualizar contrato en assets/contract_abi.json
3. Configurar red en lib/nido.dart
